You can get a brief overview of your project settings via an API endpoint. Please use the below endpoint to view your project settings.

 

Apart from project settings, this API endpoint also shows recent jobs & completed workflows.

 

Note: All the project settings are not visible through this API.

 

<circleci_url>/api/v1/project/ORG/PROJECT/settings
