Restarting deployments

Depending on the situation, Server admins may want to redeploy the various services on CircleCI Server 3.x or 4.x. 

Check deployments

The services are managed as Kubernetes deployments.

As such, we can list all deployments and their statuses via:

kubectl -n <namespace> get deployment


Restart specific deployment

You can restart a specific deployment like this:

kubectl -n <namespace> rollout restart deployment/cron-service

Alternatively, if you wish to restart ALL deployments:

kubectl -n <namespace> rollout restart deployment
