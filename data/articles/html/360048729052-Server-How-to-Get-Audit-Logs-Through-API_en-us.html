Query Server Audit Logs

Server customers can get Audit logs through UI as well as through API. If you want to use API to directly download the Audit logs,  use the below API endpoint.

 

<circleci_url>/api/v1/admin/audit-log

 

Note: Only admin may run this endpoint.
