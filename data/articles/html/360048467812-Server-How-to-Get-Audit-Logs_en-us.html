Access Server Adit Logs via the UI

CircleCI Server customers can get Audit logs in two ways through UI.

Management Console

Visit the management console and click on the "Audit Logs" tab to see the audit logs and export them.

 

 

Admin Settings

Admin users can visit the Admin Settings page while logged into CircleCI Server, and click "Audit Log". There you will see option to download the full audit logs.

 

 

 

Troubleshooting

If you are experiencing issues generating Audit Logs via the UI, you can generate Audit Logs via the API.

For further questions or additional troubleshooting, please submit a ticket to Support or write to support@circleci.com.
